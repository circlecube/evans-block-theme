# Custom Blocks


Custom blocks are basically anything beyond the core blocks. This would be any block that you build yourself and many other blocks found in various plugins or themes. For example, the jetpack plugin contains many additional custom blocks.

### Further Reading
- https://developer.wordpress.org/block-editor/how-to-guides/block-tutorial/ 
- https://jetpack.com/support/jetpack-blocks/
